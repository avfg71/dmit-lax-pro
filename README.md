## 路由情况说详细点

很多人买 CN2 GIA 就是冲着回程路由去的。Malibu 的路由情况：

**去程**（国内到美国洛杉矶）：
- 电信走 CN2（AS4809），直连不绕路
- 联通直连走 AS4837 或 CN2

**回程**（洛杉矶到国内）：

## 常见问题

**Q：Malibu 支持几台设备？**

VPS 本身不限制连接设备数，这是服务器，怎么用看你自己的配置。跑代理协议的话，同时承载多少连接取决于你的配置和实际负载，1 核 1G 对轻量个人用没问题。

**Q：流量超了会怎样？**

不停机，限速到 2 Mbps 继续跑。2 Mbps 浏览网页还行，看视频会卡，跑代理意义不大。下个月重置后自动恢复 1 Gbps。

**Q：CN2 GIA 和 CMIN2 有什么区别，Malibu 是哪个？**

CN2 GIA（AS4809）是中国电信的高端骨干网，延迟低、稳定，价格贵。CMIN2 是移动的高速国际线路，延迟略高一点，但稳定性也不错，价格便宜。Malibu 的回程三网统一走 CN2 GIA，去程电信/联通走 CN2，移动走 CMIN2。简单说，它的回程是 GIA 级别的，去程对电信用户最友好。

**Q：IP 被 GFW 封了怎么办？**

每 15 天可以申请一次免费 IP 更换，超出这个频率每次 $5。这个政策在官方 TOS 里有写，按流程提工单就行。

**Q：DMIT Malibu 值得买吗？**

如果你有国内访问优化的需求、预算有限、用途是个人节点或轻量建站，有货就买是没错的。如果需要更高配置或者不想受限量库存影响，直接看常规 Pro 系列更省心。
